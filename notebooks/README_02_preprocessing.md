### 02️⃣ Preprocessing & Standardization 🧹📊
📌 단계 개요

02 Preprocessing 단계는 본 프로젝트의 분석 기반을 만드는 핵심 단계입니다.
이 단계의 목적은 국제범죄를 추출하거나 정의하는 것이 아니라,
➡️ 국제범죄가 분석을 통해 드러날 수 있도록 데이터 구조를 정비하는 것입니다.

즉,

❌ “이 범죄는 국제범죄다”라고 선언하지 않고
⭕ “국제범죄적 성격이 나타날 수 있는 좌표계”를 구축합니다.

### 📂 입력 데이터 개요

본 단계에서는 대한민국 범죄 통계 원자료를 사용하며,
연도별로 데이터 구조가 상이한 두 구간을 다룹니다.

2005–2013년

연도별 다중 지표 컬럼 구조

예: 2005, 2005.1, 2005.2 (발생/비율/검거)

2014–2024년

Wide-format 테이블

헤더가 깨진(as-is) 형태의 CSV 포함

📌 전처리 단계에서:

발생건수만 사용

비율/검거 컬럼은 분석 대상에서 제외

### 🧩 핵심 전처리 설계 원칙
1️⃣ 범죄 분류의 평탄화 (Flattening)

원자료는 다음과 같은 계층형 분류 구조를 가집니다.

```

형법범
 └─ 강력범죄
     └─ 폭행
```

전처리 단계에서는 이를 다음과 같이 단일 관측 단위로 평탄화합니다.

형법범 / 강력범죄 / 폭행


➡️ 이를 category_raw로 정의하며,
➡️ 각 행은 서로 독립적인 관측 단위가 됩니다.

📌 이는 오류가 아니라 의도된 설계입니다.

2️⃣ category_raw vs category_std 🏷️

category_raw

원자료의 사실적 분류를 그대로 반영

계층을 결합한 문자열 형태

예: 형법범 / 재산범죄 / 사기

category_std

분석 목적을 위한 표준 범주

국제범죄 가능성을 해석하기 위한 분석 레이어

예:

- DRUG

- CYBER

- FRAUD_FIN

- IMMIGRATION

- OTHER

📌 이 단계에서
국제범죄 여부를 확정하지 않음
→ 단지 “국제범죄적 성격을 가질 수 있는 분석 축”만 제공

3️⃣ 매핑 신뢰도(confidence) 개념 🧠

각 category_raw → category_std 매핑에는
confidence 메타정보가 부여됩니다.

HIGH : 국제범죄 성격이 명확한 경우

MEDIUM : 국제범죄로 확장될 가능성이 있는 경우

LOW : 국내 범죄 성격이 강하거나 해석 여지가 큰 경우

⚠️ LOW는 오류가 아님
➡️ 조건부·맥락적 판단을 위한 의도된 상태

4️⃣ CORE / EXT 구간 분리 ⏱️

전처리 단계에서는 분석 신뢰도를 고려해 기간을 구분합니다.

CORE (2005–2023)

확정 통계 기반

구조 분석의 주 대상

EXT (2024–2025)

잠정/탐색 구간

구조적 해석은 제한적으로만 수행

### 📤 전처리 최종 산출물 (Contract Output)

전처리 단계의 결과물은 아래 3개 파일로 고정됩니다.
이후 모든 분석은 이 파일들만을 입력으로 사용합니다.

1️⃣ panel_year_category.csv 🧱

📌 핵심 분석 테이블 (Master Panel)

연도 × 표준 범주 단위의 절대 발생 건수

모든 연도에 모든 범주가 존재 (없으면 0)

주요 컬럼

- year

- category_std

- count

- source_primary

- period_flag

- notes

➡️ 규모 분석, 추세 분석의 기준 데이터

2️⃣ shares_year_category.csv 🧩

📌 구조(비중) 분석용 테이블

연도별 범죄 구성비 변화 분석

“얼마나 늘었나”보다
👉 **“구성이 어떻게 바뀌었나”**에 초점

주요 컬럼

- year

- category_std

- count

- total_count

- share

- period_flag

3️⃣ totals_year.csv 📈

📌 연도별 전체 규모 요약 테이블

해당 연도의 전체 범죄 규모

기준선(baseline) 역할

주요 컬럼

- year

- total_count

- period_flag

🚫 전처리 단계에서 의도적으로 하지 않은 것

전처리 단계에서는 아래 작업을 의도적으로 수행하지 않았습니다.

❌ 국제범죄 확정 선언

❌ 법령 기반 필터링

❌ 구조 전환(regime shift) 판단

❌ 고급 모델링 또는 클러스터링

➡️ 이는 03 / 04 단계의 분석 영역입니다.
