VISUALIZATION SPECIFICATION (TXT)
────────────────────────────────

본 문서는
「대한민국 국제범죄 구조적 변화 분석」 프로젝트에서 사용되는
시각화의 목적, 역할, 표현 수준을 단계별로 정의한다.

시각화는 단순한 결과 나열이 아니라,
분석 단계 간 논리적 전환과 구조적 통찰을 전달하는 핵심 수단으로 사용된다.

0. 시각화 품질 기준 (Quality Standard)

본 프로젝트의 시각화는 다음 기준을 충족해야 한다.

단일 메시지 중심 (One chart, one message)

다차원 정보의 집약적 표현

색상·위치·크기·궤적에 의미 부여

“예쁘다”가 아니라 “구조가 보인다”는 인상 제공

모든 시각화는
① 기존 접근의 한계를 드러내거나
② 구조적 분석의 필요성을 설명하거나
③ 국면(regime) 전환을 직관적으로 인식시키는 역할을 가진다.

Phase 1 – 기존 접근 기반 시각화 (Baseline Visualization)
목적

국제범죄를 건수·트렌드 중심으로 바라보는 기존 분석 방식을 명확히 제시하고,
동시에 해당 접근이 가지는 설명 한계를 자연스럽게 드러낸다.

[V1] 국제범죄 전체 발생 건수 시계열

유형: Line Chart

축 구성:
· X축: 연도 (2005–2025)
· Y축: 국제범죄 전체 발생 건수

표현 포인트:
· 단일 선, 단색
· 불필요한 주석 최소화

전달 메시지:
“국제범죄는 특정 시기에 증가 또는 감소하는 경향을 보인다.”

한계 노출:
· 왜 증가했는지
· 어떤 범죄가 주도했는지는 설명 불가

[V2] 국제범죄 유형별 발생 건수 추이

유형: Multi-line Chart

축 구성:
· X축: 연도
· Y축: 발생 건수
· 선: 범죄 유형별

표현 포인트:
· 주요 4~6개 범죄 유형만 선택
· 색상은 범죄 유형 고정

전달 메시지:
“범죄 유형별로 서로 다른 증감 패턴이 존재한다.”

한계 노출:
· 개별 유형은 보이지만
· 전체 범죄 환경의 ‘구성’은 직관적으로 보이지 않음

[V3] 국제범죄 유형별 구성비 변화

유형: Stacked Area Chart

축 구성:
· X축: 연도
· Y축: 구성비(%)

표현 포인트:
· 건수가 아닌 비중 강조
· 색상 대비 최소화

전달 메시지:
“국제범죄는 단순 증가가 아니라 구성비가 변화하고 있다.”

Phase 전환용 문장:
“그러나 구성비 변화 역시 연도별 나열에 그치며,
구조적 전환 시점을 명확히 설명하지는 못한다.”

➡️ Phase 2로 넘어가기 위한 결정적 시각화

Phase 2 – 구조 기반 시각화 (Advanced Structural Visualization)
목적

국제범죄를 다변량 구조로 재정의하고,
구조의 이동·전환·국면을 직관적으로 인식 가능하게 한다.

[V4] 구조 변화 탐지 시계열 (Change Point Overlay)

유형: Line Chart + Vertical Markers

구성:
· V1 또는 V3 그래프 위에
· 구조적 전환점(change point)을 수직선으로 표시

전달 메시지:
“범죄 구조는 특정 시점에서 체계적으로 변화했다.”

고급 포인트:
· 시각적으로 Phase 1 그래프와 연결
· ‘트렌드 변곡점’이 아니라 ‘구조 변곡점’임을 강조

[V5] 구조 벡터 평균 비교 (전환 전·후)

유형: Grouped Bar Chart

구성:
· X축: 범죄 유형
· Y축: 평균 구성비
· 그룹: 전환 전 / 전환 후

전달 메시지:
“구조 전환은 특정 범죄 유형의 상대적 비중 변화로 설명된다.”

고급 포인트:
· 절대 수치 ❌
· 상대 구조 ⭕

[V6] 구조 궤적 시각화 (PCA / UMAP Trajectory)

유형: 2D Scatter + Trajectory Line

구성:
· 점: 연도
· 선: 연도 순 연결
· 색상: 국면(regime)

전달 메시지:
“국제범죄 구조는 시간에 따라 연속적으로 이동해 왔다.”

프로젝트 핵심 시각화

README의 대표 이미지 후보

[V7] 국면(regime)별 구조 요약

유형: Regime-wise Average Structure Bar Chart

구성:
· X축: 범죄 유형
· Y축: 평균 구성비
· 패널: Regime 1 / 2 / 3

전달 메시지:
“각 국면은 서로 다른 범죄 구조적 특성을 가진다.”

Phase 3 – 확장 구간 시각화 (Exploratory Extension)
[V8] 2024–2025 구조 투영

유형: PCA 공간 투영 Scatter

구성:
· 기존 연도: 회색
· 2024–2025: 강조 색상

전달 메시지:
“최근 연도는 기존 국면 중 특정 구조와 가장 유사한 위치에 놓인다.”

주의 문구 필수:
“본 시각화는 탐색적 관측이며 구조적 결론을 의미하지 않는다.”

요약 – 시각화 설계의 핵심 논리

Phase 1 시각화는 한계를 보여주기 위해 존재

Phase 2 시각화는 구조를 이해시키기 위해 존재

모든 시각화는 다음 질문에 답해야 한다:
“이 그림이 없으면, 독자는 무엇을 이해하지 못하는가?”
